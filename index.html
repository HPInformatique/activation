Bienvenue dans le centre d'activation de produits HPI

Informations :

Pour continuer l'activation de votre produit HPI, nous devons vérifier que vous faites partie de l'espace membre de HP Informatique. Pour ce faire, il vous faut obtenir un code (étape 1 de l'activation) puis entrer le code dans le champ de confirmation (étape 2). Si vous n'êtes pas connecté(e) à votre compte Google sur cet appareil, il se peut qu'une fenêtre s'ouvre avec la page de connexion de Google. Cela permet de vous authentifier sur le site de HPI. Si vous ne faites pas parti(e) de l'espace membre de HPI, vous tomberez sur une page 404 erreur.
